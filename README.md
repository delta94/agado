## Installation
```shell
$ npm install
```

## Building
To compile TypeScript source code:
```shell
$ npm run build
```

## Starting the server
```shell
# The source code must have been compiled (with npm run build)
$ npm start
```

## Testing
```shell
# Run normal tests
$ npm run test

# Run tests with code coverage collection
$ npm run test-coverage
```
